NAME

Alambic::Plugins::JiraIts - Retrieves issue tracking data and metrics from a Jira server.

DESCRIPTION

Alambic::Plugins::JiraIts Retrieves issue tracking data and metrics from a Jira Server.

Parameters:

  • jira_open_states A list of status names considered as open in the workflow, coma-separated.

  • jira_passwd The password of the user for the connection to the Jira server.

  • jira_project The Identifier of the project within the Jira instance, e.g. ALX.

  • jira_url The URL of the Jira server, e.g. https://tracker.openattic.org.

For the complete description of the plugin see the user documentation on the web site: https://alambic.io/Plugins/Pre/Jira.html.
