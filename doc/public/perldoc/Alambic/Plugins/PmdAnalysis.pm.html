NAME

Alambic::Plugins::PmdAnalysis - A plugin to analyse PMD results and help improve the configuration of the tool.

DESCRIPTION

Alambic::Plugins::PmdAnalysis helps people better understand the results of the PMD tool.

Parameters:

  • url_pmd_xml An URL that points to the PMD XML configuration file.

  • url_pmd_conf An URL that points to the PMD XML results file.

For the complete configuration see the user documentation on the web site: https://alambic.io/Plugins/Pre/PmdAnalysis.html.
