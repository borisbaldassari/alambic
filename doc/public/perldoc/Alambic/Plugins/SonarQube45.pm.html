NAME

Alambic::Plugins::SonarQube45 - A plugin to fetch information from a SonarQube 4.5.x instance.

DESCRIPTION

Alambic::Plugins::SonarQube45 retrieves information from a SonarQube 4.5.x instance.

Parameters:

  • SonarQube server URL The URL of the SonarQube server, e.g. https://sonar.eclipse.org or http://localhost:9000.

  • Project ID The ID of the project within the SonarQube instance, e.g. log4j or org.polarsys:org.polarsys.capella.

For the complete configuration see the user documentation on the web site: https://alambic.io/Plugins/Pre/SonarQube45.html.
