NAME

Alambic::Plugins::EclipsePmi - A plugin to fetch information from the Eclipse PMI repository.

DESCRIPTION

Alambic::Plugins::EclipsePmi retrieves information from the Eclipse PMI repository.

Parameters:

  • Eclipse project ID - e.g. modeling.sirius or tools.cdt.

For the complete configuration see the user documentation on the web site: https://alambic.io/Plugins/Pre/EclipsePmi.html.
