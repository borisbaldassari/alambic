NAME

Alambic::Model::RepoFS - Interface to all file-system-related actions and information defined in Alambic.

SYNOPSIS

    my $repofs = Alambic::Model::RepoFS->new();
    
    $repofs->delete_project('modeling.sirius');

DESCRIPTION

Alambic::Model::RepoFS provides autility methods for all operations related to file system: read and copy files, delete projects, etc.
