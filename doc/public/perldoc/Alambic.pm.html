NAME

Alambic - An open-source platform and service for the management and visualisation of software engineering data.

SYNOPSIS

Start the application in prefork mode (production mode, multi-threaded):

  $ hypnotoad bin/alambic
  Server available at http://127.0.0.1:3010
  $

Start the application in daemon mode (development mode, automatic reload of files):

  $ morbo bin/alambic
  Server available at http://127.0.0.1:3000

Start a worker thread (used for long-running tasks, e.g. project analysis):

  $ bin/alambic minion worker

DESCRIPTION

Alambic is an open-source framework and service for the management and visualisation of software engineering data.

The official web site with complete documentation is http://alambic.io.

Project is developed on BitBucket at https://bitbucket.org/BorisBaldassari/alambic.

See "COMMANDS" in Alambic::Commands for a list of commands that are available.
