NAME

Alambic::Model::Plugins - Interface to all plugins defined in Alambic.

SYNOPSIS

    my $plugins = Alambic::Model::Plugins->new();
    
    my $plugin_so = $plugins->get_plugin('StackOverflow');
    my $list_pre = $plugins->get_list_plugins_pre();

DESCRIPTION

Alambic::Model::Plugins provides a complete interface to the Plugins management of Alambic, with ability to get lists of plugins according to their type or ability. Also provides access to the plugin objects themselves.

METHODS

new()

    my $plugins = Alambic::Model::Plugins->new();

Create a new Alambic::Model::Plugins object and initialises it with the list of plugins on the file system (i.e. Alambic/Plugins/*.pm).
