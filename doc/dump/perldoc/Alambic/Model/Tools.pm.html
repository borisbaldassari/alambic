NAME

Alambic::Model::Tools - A class to manage, install and execute external tools in Alambic. Provides a common interface to all tools plugins.

SYNOPSIS

    my $tools = Alambic::Model::Tools->new();    
    my $list = $tools->get_list_all();

DESCRIPTION

Alambic::Model::Tools provides a common interface to a series of tools used in Alambic. Tool plugins can provide the following methods:

  • install provides an automatic install of the tool.

  • version returns the version of the tool.

  • test is a self-diagnostic method for tools auto-test.

  • specific methods can be provided as listed in the plugin's configuration hash.

METHODS

new()

    my $tools = Alambic::Model::Tools->new();

Creates a new Alambic::Model::Tools object to interact with the various tools used within Alambic.

get_list_all()

    my $list = $tools->get_list_all();

Get the list of tools recognised by Alambic, as an array ref of tool plugin ids (strings).

    [
      'git',
      'r_sessions'
    ];

get_tool()

    my $git = $tools->get_tool('git');
    my $r = $tools->get_tool('r_sessions');

Get a reference to the tool object (e.g. Alambic::Tools::Git).
