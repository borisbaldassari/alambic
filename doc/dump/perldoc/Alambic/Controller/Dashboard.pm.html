NAME

Alambic::Controller::Dashboard - Routing logic for the Alambic dashboard.

SYNOPSIS

Routing logic for all dashboard-related actions in the Alambic web ui. This is automatically called by the Mojolicious framework.

METHODS

display_summary()

Main page for project dashboard.

display_project()

Secondary dashboard pages for a specific project.

display_history()

Main page for project history.

display_plugins()

Display figures and visualisations of plugins.

display_figures()

Is used only for backward compatibility and robustness. R reports often use figures/ by default for the generated plots. We want to support that for people developing R books.
