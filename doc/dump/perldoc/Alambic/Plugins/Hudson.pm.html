NAME

Alambic::Plugins::Hudson - A plugin to fetch information from the Eclipse PMI repository.

DESCRIPTION

Alambic::Plugins::Hudson retrieves information from a Hudson instance.

Parameters:

  • Hudson URL A link to the root of the project's Hudson instance.

For the complete configuration see the user documentation on the web site: https://alambic.io/Plugins/Pre/Hudson.html.
