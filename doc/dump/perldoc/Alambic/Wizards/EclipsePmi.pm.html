NAME

Alambic::Wizards::EclipsePmi - A wizard plugin to add a project from the Eclipse forge.

DESCRIPTION

Alambic::Model::Plugins provides a way to easily initialise a project from the Eclipse forge.

Parameters:

  • Eclipse project ID - e.g. modeling.sirius or tools.cdt.

Plugins automatically initialised with this wizard:

  • Hudson - e.g. modeling.sirius or tools.cdt.
