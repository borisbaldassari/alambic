Alambic releases

Alambic 3.3.3

Date: 2017-12-05

Changelog:

  • Minor bug fixes.
  • New Proxy feature for all plugins, contributed by Thales Group.
  • Support for Mojo::Minion 8 new API.
  • Updated documentation, available on https://alambic.io.

If one intends to run Alambic with a recent version of Minion (>= 8), then this release (or greater) is needed.
